Extract Components


The Extract Components action is used for automatically populating the configuration part of how the components should be displayed in the MSC view, i.e. the Component Visual Order, by extracting the names of all test components used within a single TITAN log file. If the component name can not be determined the component reference is added. Extracted components are always added on project level.

  1. Select a file with the extension log in the Navigator.

  2. Select the action Extract Components in the pop-up menu.

    The log file is processed and the component names, or if the name can not be determined the reference, are extracted. Depending on the user setting of Principle when adding components in the General Settings, the extracted components are replacing or appended to the existing components in Components Visual Order list.
    If the user setting Open properties after extracting components in the General Settings is set the project properties dialog automatically opens after the completion of the extraction.
